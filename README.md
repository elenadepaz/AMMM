# AMMM
Repository for the Algorithmic Methods for Mathematical Models course, part of the MIRI Master's program at UPC (Fall 2025). Contains implementations, exercises, and projects related to optimization, numerical methods, and algorithmic modeling.

## Course Contents

- **Optimization Techniques**: Linear and nonlinear programming, metaheuristic algorithms
- **Mathematical Modeling**: Problem formulation, constraint definition, solution strategies
- **Practical Implementation**: CPLEX solutions, algorithm comparison, performance analysis
